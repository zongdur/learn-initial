点击要找到的元素，ctrl+F，可以对页面里的数据进行查找


（1）css定位器定位元素
类选择器：. class
id选择器： #  加上id值
父类选择器  子类选择器：  类名   空格   类名
标签选择器  ：标签名


（2）Xpath定位元素
1.绝对路径（通过“/”开头）
2.相对路径（“//”开头）
【1】相对路径加索引  //form/span【1】
【2】相对路径加上属性值  //from【@class="id_name"】
【3】相对路径加上通配符 //form/*[@*="bdsug bdsug-new new-pmd"]
【4】相对路径加上文本  //a[text()="新闻"]


css定位方式想对于Xpath定位方式更高效
