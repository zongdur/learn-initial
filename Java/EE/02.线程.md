死锁的成因：
1.互斥使用（锁的基本特性）：当一个线程加上一把锁后，另一个线程想要获取这把锁，就只能阻塞等待
2.不可抢占（基本特性）：锁被线程1拿到后，线程二要获取锁，只能等待，不能强行抢占
3.请求保持：一个线程获取多把锁（获取锁一后，再获取锁二）
4.循环等待：关系冲突，形成闭环
